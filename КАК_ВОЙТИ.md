# 🔐 Как войти на сайт через Telegram

## Быстрая инструкция

1. **Откройте Telegram бот**: [@WeatherNewsHubot](https://t.me/WeatherNewsHubot)

2. **Получите код**:
   - Отправьте команду `/login`
   - Или нажмите кнопку "🔐 Вход на сайт"
   - Бот пришлёт вам 6-значный код

3. **Войдите на сайт**:
   - Откройте http://localhost:5173/login
   - Нажмите "У меня есть код"
   - Введите код из Telegram
   - Нажмите "Войти"

## ⚠️ Важно

- Код действителен **5 минут**
- Код **одноразовый** (после использования удаляется)
- Если код не работает, запросите новый через `/login`

## 🚀 Запуск сервисов

Если сервисы не запущены:
```bash
./start.sh
```

Или вручную:
```bash
# Терминал 1: API
cd bot && source venv/bin/activate && python3 api_server.py

# Терминал 2: Бот
cd bot && source venv/bin/activate && python3 main.py

# Терминал 3: Сайт
cd front && npm run dev
```

## 📱 Доступные адреса

- **Сайт**: http://localhost:5173
- **API**: http://localhost:8000
- **API Docs**: http://localhost:8000/docs
- **Telegram бот**: [@WeatherNewsHubot](https://t.me/WeatherNewsHubot)
