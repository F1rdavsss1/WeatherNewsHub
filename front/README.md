# WeatherNews Hub - Frontend

Современный веб-сайт для отображения погоды и новостей с возможностью персонализации.

## 🚀 Технологический стек

- **React 18+** - UI библиотека
- **TypeScript** - типизация
- **Vite** - сборщик
- **Tailwind CSS** - стилизация
- **React Query** - управление серверным состоянием
- **Zustand** - управление клиентским состоянием
- **React Router v6** - роутинг
- **Framer Motion** - анимации
- **React Hook Form + Zod** - формы и валидация
- **React Hot Toast** - уведомления
- **Axios** - HTTP клиент
- **date-fns** - работа с датами

## 📦 Установка

```bash
# Установка зависимостей
npm install

# Копирование .env файла
cp .env.example .env

# Настройка переменных окружения
# Отредактируйте .env файл
```

## 🔧 Переменные окружения

```env
VITE_API_URL=http://localhost:8000
VITE_TELEGRAM_BOT_NAME=weathernews_bot
```

## 🏃 Запуск

```bash
# Режим разработки
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр продакшен сборки
npm run preview
```

## 🎨 Основные функции

### Главная страница
- Отображение текущей погоды
- Прогноз на неделю
- Лента новостей с фильтрацией
- Поиск городов с автодополнением

### Авторизация
- Вход через Telegram (в разработке)
- Вход по Email/паролю

### Личный кабинет
- Управление профилем
- Избранные города
- Настройки уведомлений
- Персонализация темы

## 🌐 API Интеграция

Приложение интегрируется с FastAPI бэкендом:

- `GET /weather/current?city={city}` - текущая погода
- `GET /weather/forecast?city={city}&days={days}` - прогноз
- `GET /news?category={cat}&country={country}` - новости
- `POST /auth/login` - авторизация
- `GET /user/profile` - профиль пользователя

## 🎨 Темизация
